## 价值模型（卡牌设计参考）

| 费用 | 价值 |
|---|---|
| 0 | 0 |
| 1 | 8 |
| 2 | 14 |
| 3 | 18 |

**基础效果价值**：子弹+1=6 / 波数+1=12 / 弹射+1=2 / 穿透+1=4 / 火焰+1=约 6 / 追踪=4 / 召唤物=8~12

**修饰符**：展露 ×3 / 洗入每张 -7 / 弃置触发 -7 / 连击需求 -6-(0.5×额外要求) / 随机效果平均 -1

## 召唤系统规则

- 召唤物只在**敌方回合**行动（移动 / 发射）；它们发射的子弹照常每帧飞
- 敌人 AI 会找场上**最近的我方单位**（玩家 + 召唤物）
- 每个敌方回合开始，所有友方召唤物 **HP -1**（工兵 decayRate=0 例外，不衰减）
- 持续召唤卡按敌方回合计时（如召唤增援 3 回合）
- **召唤物子弹继承玩家 buffs**（主卡 hooks + 展露 hooks + PreActive）：凝视主卡 → 召唤物子弹也 +1；火焰主卡 → 召唤物子弹带火焰

## 波次系统

- 第 0 波在按 Enter 后立即生成
- 每 m 回合 spawn 一波，m = `[5,5,4,4,3,3,3,2]`（按 shopLevel 1-8）
- 每波价值 n = `floor(8 + 3w + 0.3w²)`（w=0:8, w=5:30, w=10:59）
- 用背包问题随机填充：从 SPAWN_POOL[shopLevel] 中按敌人 value 随机组合
- **奖励回合**：场上无敌人 + 波次未到 → 3 个金球（不动、不能 kill），击中按指数递减给金币

## XP 曲线

- 每级所需经验：`min(100, floor(6 + 4(L-1) + 0.6(L-1)²))`
- L=1→2: 6 XP / L=5→6: 31 / L=10→11: 90 / L=11+: 100（封顶）
- 升级 → 入队 pendingShops（**不立即开商店**）；回合结束时弹出商店
