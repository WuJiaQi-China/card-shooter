# 卡牌射击 · Web 复刻 (MVP)

Unity 项目 `C:\项目\卡牌射击` 的 HTML5 Canvas 复刻版，单文件零依赖。

## 运行

**最简单**：双击 `index.html` 即可。

如果浏览器策略阻止本地脚本（极少见），用任意静态服务器跑：

```powershell
# Python 3 (推荐)
python -m http.server 8765
# 然后浏览器访问 http://localhost:8765
```

## 操作（鼠标主控，与 Unity 原版一致）

| 键 | 功能 |
|---|---|
| 鼠标移动 | 炮台瞄准 |
| 鼠标 **左键** | 使用最左侧手牌 |
| 鼠标 **右键** | 使用最右侧手牌 |
| 滚轮 **上** | 弃掉最左侧 |
| 滚轮 **下** | 弃掉最右侧 |
| `Space` | **结束回合**（回合制） |
| `F` | 连携（左 + 右 + 主卡一起打出） |
| `Enter` | 开战 / 阵亡后重开 |

胜利条件：击杀 5 个敌人（接触自爆不算）。

## 回合制

- **水晶不自动回复**：每个玩家回合开始时一次性回满 10/10。
- 玩家回合期间怪物**冻结**（不移动、不攻击）。
- 按 `Space` **结束回合** → 进入敌方回合（默认 2 秒）：
  - 怪物开始移动 / 接触自爆扣血。
  - 玩家可以**同步操作**（继续射击 / 弃牌），但水晶用完就用完。
- 敌方回合结束 → 自动回到玩家回合 + 水晶满。

### 设置：水晶用尽自动结束回合
HUD 右上角的复选框。勾上后水晶 = 0 时自动触发 `endPlayerTurn`，省一次手动按 `Space`。

### 接触行为
怪物撞到炮台 = **一次性扣血 + 消失**（自爆模型，不再贴脸蹭血）。  
自爆 **不计入** 击杀数，只有用子弹打到 HP=0 才算 kill。

## 升级流程（每场胜利触发）

1. **3 选 1** —— 胜利后弹出 3 张随机新卡，点击其中一张选中
2. **替换** —— 进入背包面板，点击 10 张背包卡中任意一张，用新卡替换它
3. **调整**（可选）—— 替换后可继续操作：
   - **右键** 任意卡 → 设为主卡（与主卡位 0 交换）
   - **拖拽** 一张卡到另一张上 → 交换位置
4. **继续游戏** —— 点击「继续游戏」按钮开下一场战斗

## 复刻范围

跟原 Unity 项目 1:1 对齐的部分：

### Hook 系统（6 阶段，按 priority 升序）
- `PreActive` —— 模板子弹改属性（如「增弹」+2、「鱼目混珠」+2 弹/+2 穿）
- `Spawned` —— 每颗克隆独立决策（「不稳定增弹」50% 哑弹的唯一时机）
- `HitWall` —— 「鱼目混珠」从 pen 借 bound
- `HitEnemy` —— 「鱼目混珠」从 bound 借 pen
- `Destroyed` —— 清理
- `PostActive` —— 一波打完

Hook 实例可被多颗克隆共享 → **必须无状态**（per-bullet 状态写到 `Bullet` 字段）。

### 4 张代表卡
- **增弹**（Card_AddTwoBullets）：PreActive +2 bulletCount
- **鱼目混珠**（Card_SharedBouncePierce）：PreActive +2/+2，HitWall/HitEnemy 互借
- **不稳定增弹**（Card_AddThreeUnstable）：PreActive +3，Spawned 每颗 50% attack=0
- **凝视**（Card_Gaze）：使用 +1；展露光环每次发射免费 +1

### 战斗状态机
`Idle → PreBattle → Battle → (PostBattle | Reward → Inventory) → Idle`

`Battle` 内还有正交的 `turn`：`player` ↔ `enemy`。

### 卡组规则
- `bag[0]` 是主卡（常驻正面、参与每次普通使用、双重扣费 + 双重 Hook）
- 手牌横排，**最左 + 最右是边缘卡** —— 自动翻为正面、触发 `OnReveal`
- 卡面切换 face-up ↔ face-down 触发 **CSS 3D 翻转动画**（DOM 按 cardRef 复用以让 transition 生效）
- 用过的牌进弃牌堆，手牌空时从弃牌堆补到上限

### 连击
- 同侧连续使用 → `combo++`
- 换侧 / 弃牌 / 战斗起止 → 清零
- 连携（`F`）→ `combo+1`、lastSide=`any`（下一张任意侧延续）
